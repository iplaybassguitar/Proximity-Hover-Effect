# Proximity-Hover-Effect
Small dots light up as you hover your mouse over.

A lightweight, high-performance, interactive background effect built with Vanilla JavaScript and CSS Grid. This project creates a responsive field of dots that "light up" based on the proximity of your mouse cursor.

 - Proximity Interaction: Dots react dynamically to the cursor’s position using distance calculations.
 - Uses CSS to fill the screen size.
 - Built entirely on HTML, CSS and JavaScript.
